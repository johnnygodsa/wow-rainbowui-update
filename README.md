# 彩虹UI自動更新器

- 專為魔獸世界(World of Warcraft)打造的一鍵式彩虹UI更新工具
- 提供繁體中文介面與安裝程式,讓非技術背景的玩家也能輕鬆使用
- 自動同步最新的UI組件,無需手動下載與解壓縮

## 主要功能

- **一鍵更新** - 自動偵測魔獸世界安裝路徑,點擊按鈕即可完成UI更新
- **自動路徑偵測** - 智能掃描常見的WoW安裝位置,免除手動設定困擾
- **定時自動更新** - 可設定1-24小時間隔自動檢查並更新UI
- **內建Git工具** - 整合MiniGit(約55MB),無需額外安裝Git或相關工具
- **即時日誌顯示** - 完整記錄更新過程,方便排查問題

## 系統需求

- **作業系統** - Windows 10/11
- **必要條件** - 已安裝魔獸世界(正式服)
- **硬碟空間** - 建議預留500MB以上空間

## 安裝方式

### 使用安裝程式(推薦)

- 下載最新版本的 `wow-rainbowui-update-Setup.exe`
- 執行安裝程式,按照畫面指示完成安裝
- 安裝程式會自動偵測魔獸世界資料夾並建議安裝位置
- 建議安裝於魔獸世界根目錄下,例如: `D:\World of Warcraft\wow-rainbowui-update`

### 安裝程式特色

- **繁體中文介面** - 完整中文化的安裝流程
- **智能路徑建議** - 自動掃描註冊表與常見安裝位置
- **包含MiniGit** - 安裝檔已內建所有必要工具
- **可選桌面捷徑** - 安裝時可選擇是否建立桌面捷徑

## 使用說明

### 首次使用

- 啟動更新器後,若尚未設定路徑,會自動開啟檔案選擇對話框
- 選擇魔獸世界安裝資料夾中的 `World of Warcraft Launcher.exe`
- 程式會自動識別並設定正確的 `_retail_\Interface\AddOns` 路徑

### 執行更新

- 點擊 **「開始更新」** 按鈕
- 更新器會自動從GitHub下載最新的彩虹UI版本
- 首次更新需下載完整檔案,後續更新僅下載變更部分
- 更新完成後,重新啟動魔獸世界即可使用最新UI

### 定時更新設定

- 勾選 **「啟用定時更新」** 選項
- 設定更新間隔時間(1-24小時)
- 程式會在背景定時檢查並自動更新UI

## 技術架構

### 更新機制

- 使用Git進行版本控制與增量更新
- 透過淺層克隆(shallow clone)加速下載
- 僅同步變更的檔案,節省頻寬與時間

### 資料來源

- **官方倉庫** - [WOWRainbowUI/RainbowUI-Retail](https://github.com/WOWRainbowUI/RainbowUI-Retail)
- **更新分支** - master
- **更新頻率** - 跟隨彩虹UI作者更新

## 常見問題

### 更新失敗怎麼辦?

- 檢查網路連線是否正常
- 確認防火牆或防毒軟體未阻擋程式
- 查看日誌區域的錯誤訊息
- 嘗試刪除安裝目錄下的 `Repo` 資料夾後重新更新

### 可以更新測試服或經典版嗎?

- 目前僅支援正式服(Retail)版本
- 測試服與經典版請使用對應的更新工具

### 需要GitHub帳號嗎?

- 基本功能無需帳號即可使用
- API呼叫受限於未認證用戶的速率限制(60次/小時)
- 若需更頻繁更新,可設定GitHub Token提升至5000次/小時
