# 🌈 魔獸世界彩虹 UI 自動更新器

## ✨ 專案簡介

這是一款專為**彩虹 UI 整合包**打造的獨立更新工具，提供比官方方案更便捷的更新體驗。

## 🎯 核心特色

### 更簡單的更新方式
- ❌ 不需要安裝 GitHub Desktop
- ❌ 不需要複雜的設定步驟
- ✅ 只需選擇魔獸世界執行檔，自動定位安裝路徑
- ✅ 一鍵檢測並更新所有 UI 插件

### 智能更新機制
- 直接從 [WOWRainbowUI/RainbowUI-Retail](https://github.com/WOWRainbowUI/RainbowUI-Retail) 官方倉庫獲取最新版本
- 自動比對本地與雲端版本差異，只下載需要更新的檔案
- 支援增量更新，節省頻寬與時間
- 智能檢測缺失的插件資料夾並自動補齊

### 效能優化
- 支援 **GitHub Token**，將 API 速率限制從 60 次/小時提升至 **5000 次/小時**（快 83 倍）
- 多線程下載，提升更新速度
- 實時顯示下載進度、速度與剩餘時間
- 友善的繁體中文介面與操作提示

## 📦 與官方方案的比較

| 特性 | 官方 GitHub Desktop 方案 | 本更新器 |
|:------|:------------------------|:---------|
| 安裝複雜度 | 需安裝 GitHub Desktop | 免安裝，開箱即用 |
| 設定難度 | 需設定 Git Clone | 自動定位路徑 |
| 更新速度 | 受網路限制 | 支援 Token 加速 |
| 檔案大小 | 下載完整倉庫 | 僅下載需要的檔案 |
| 適用對象 | 進階使用者 | 所有玩家 |

## 🚀 使用方法

1. 下載並執行 `InstallerWinForms.exe`
2. 點擊「選擇」按鈕，找到 `World of Warcraft Launcher.exe`
3. 點擊「更新」按鈕，等待自動完成
4. （選用）設定 GitHub Token 以加速下載

## 💡 適合對象

- 不想安裝 GitHub Desktop 的玩家
- 追求簡單便捷更新流程的使用者
- 希望節省下載時間與頻寬的玩家
- 需要頻繁更新 UI 的重度使用者

## 📌 注意事項

- 本工具為非官方第三方更新器
- 更新來源與官方倉庫完全相同，確保安全性
- 建議首次使用前備份現有插件設定
