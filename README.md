# 🌈 彩虹UI自動更新器

- **一鍵更新魔獸世界彩虹UI插件集合的專用工具**
- **使用 Git 技術自動同步最新版本,無需手動下載**
- **內建繁體中文安裝程式,完全免費開源**

---

## 📋 功能特色

- ✨ **自動更新** - 一鍵檢查並安裝最新版彩虹UI
- ⏰ **定時更新** - 可設定每隔數小時自動檢查更新(1-24小時)
- 📊 **即時進度** - 清楚顯示下載進度、速度與預估完成時間
- 🔍 **智慧比對** - 自動偵測需要更新的UI資料夾
- 💾 **節省流量** - 採用增量更新技術,只下載變更的檔案
- 🖥️ **系統匣最小化** - 可縮小至系統匣背景執行
- 📝 **詳細日誌** - 完整記錄更新過程,方便問題排查

---

## 💿 安裝方式

### 方式一:使用安裝程式(建議) ⭐

1. 下載最新版安裝程式 `wow-rainbowui-update-Setup.exe`
2. 執行安裝程式,選擇安裝路徑
3. **建議安裝在魔獸世界根目錄下**,例如:
   - `D:\Program Files\World of Warcraft\wow-rainbowui-update`
4. 安裝程式會自動配置 MiniGit 環境
5. 安裝完成後直接啟動即可使用

### 方式二:手動安裝

1. 下載程式壓縮檔並解壓縮
2. 確保資料夾內包含以下檔案:
   - `彩虹UI更新器.exe`
   - `彩虹UI更新器.dll`
   - `MiniGit` 資料夾(完整)
3. 雙擊 `彩虹UI更新器.exe` 啟動

## 🎮 使用說明

### 首次使用

1. 啟動更新器後,點擊 **「選擇」** 按鈕
2. 瀏覽至 **魔獸世界安裝目錄**
3. 選擇 `World of Warcraft Launcher.exe`
4. 點擊 **「點此更新」** 按鈕開始安裝UI

### 日常更新

- 每次開啟魔獸前,執行更新器並點擊更新按鈕
- 程式會自動檢查並下載最新版UI
- 更新完成後可直接啟動遊戲

### 定時更新設定

- 勾選 **「啟用定時更新」**
- 設定更新間隔(1-24小時)
- 最小化程式至系統匣,自動在背景執行

## 📦 技術說明

### 系統需求

- **作業系統**: Windows 7 / 10 / 11
- **.NET Runtime**: .NET 6.0 或更高版本
- **網路連線**: 需連接網際網路下載更新
- **磁碟空間**: 建議至少 500MB 可用空間

### 更新來源

| 項目 | 說明 |
|------|------|
| **資料來源** | GitHub 官方倉庫 |
| **更新頻率** | 跟隨彩虹UI作者發布頻率 |
| **倉庫位置** | [WOWRainbowUI/RainbowUI-Retail](https://github.com/WOWRainbowUI/RainbowUI-Retail) |
| **分支** | master |

### 內建工具

- **MiniGit** - 輕量化 Git 工具(約 55MB)
- **無需安裝額外軟體** - 所有依賴項目已整合

## ⚙️ 進階設定

- 設定檔位置:`rainbow_config.json`
- 可手動編輯調整以下參數:

| 參數 | 說明 | 預設值 |
|------|------|--------|
| `AutoUpdateEnabled` | 是否啟用定時更新 | `false` |
| `AutoUpdateInterval` | 更新間隔(小時) | `1` |
| `ParallelDownloads` | 並行下載數 | `3` |
| `ConnectionTimeout` | 連線逾時(秒) | `30` |

## ❓ 常見問題

### Q: 更新失敗怎麼辦?

- 檢查網路連線是否正常
- 查看日誌區的錯誤訊息
- 嘗試重新執行更新

### Q: 更新器會覆蓋我的UI設定嗎?

- 不會!更新器只會更新 AddOns 資料夾內的UI檔案
- 您的 WTF 資料夾(存放設定)不會被影響

### Q: 可以改用其他UI倉庫嗏?

- 可以,手動編輯 `rainbow_config.json` 中的 `RepoFullName` 參數

### Q: 關閉視窗後程式還在執行?

- 關閉時會詢問是否最小化到系統匣或完全關閉
- 可在系統匣圖示右鍵選單中完全退出
